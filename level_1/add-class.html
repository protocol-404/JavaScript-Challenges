Ajouter une classe CSS à un élément

Retour

Description

Ajoutez une classe CSS à un élément lors d'un clic.

Solution


const targetElement = document.getElementById('targetElement');
const addClassButton = document.getElementById('addClassButton');

addClassButton.addEventListener('click', () => {
    targetElement.classList.toggle('highlight');

    const hasClass = targetElement.classList.contains('highlight');
    addClassButton.textContent = hasClass ? 'Retirer la classe' : 'Ajouter la classe';
});

Démo

Élément Cible

Cet élément changera de style lorsque vous cliquerez sur le bouton ci-dessus. La classe CSS 'highlight' sera ajoutée ou retirée à chaque clic.
