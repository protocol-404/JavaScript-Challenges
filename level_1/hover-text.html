Change element text on mouse hover

Back

Description

Change the text of an element when the mouse hovers over it.

Solution


const testHoverChange = document.getElementById('testHoverChange');

testHoverChange.addEventListener('mouseover', () => {
    testHoverChange.textContent = 'Text changed on mouse hover!';
});
            
testHoverChange.addEventListener('mouseout', () => {
    testHoverChange.textContent = 'Hover over this text to change it.';
});
