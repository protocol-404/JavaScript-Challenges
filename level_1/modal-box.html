Créer une boîte modale qui s'affiche et se ferme avec un bouton

Retour

Description

Ajoutez un bouton qui affiche une boîte modale. Ajoutez un autre bouton dans la boîte pour la fermer.

Solution


const modalBox = document.getElementById('modalBox');
const boxButton = document.getElementById('boxButton');
const modalCloseButton = document.getElementById('modalCloseButton');
            
boxButton.addEventListener('click', () => {
    modalBox.classList.remove('hidden');
});
            
modalCloseButton.addEventListener('click', () => {
    modalBox.classList.add('hidden');
});
            
modalBox.addEventListener('click', (e) => {
    if (e.target === modalBox) {
        modalBox.classList.add('hidden');
    }
});
