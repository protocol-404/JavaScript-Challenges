Alterner la visibilité d'un élément lors d'un clic

Retour

Description

Créez un bouton qui montre ou cache un élément lors d'un clic.

Solution


const toggleButton = document.getElementById('toggleButton');
const toggleElement = document.getElementById('toggleElement');

toggleButton.addEventListener('click', () => {
    toggleElement.classList.toggle('hidden');
    
    const isHidden = toggleElement.classList.contains('hidden');
    toggleButton.textContent = isHidden ? 'Afficher l\'élément' : 'Cacher l\'élément';
});

Démo

Élément Togglable

Cet élément peut être affiché ou caché en cliquant sur le bouton ci-dessus. C'est un exemple simple de manipulation de la visibilité d'un élément en JavaScript.
